# template-go

> clean template for go project

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
$ yarn global add sao
# or
$ npm i -g sao
```

### From git

```bash
$ sao zcong1993/template-go my-project 
# first time
$ sao zcong1993/template-go my-project --install
```
