# template-go

> SAO template for golang

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From git

```bash
sao zcong1993/template-go my-project
```

### From npm

```bash
sao go my-project
```
